# StudentUnionWebsite

Website of Student Union,Nexus(2025-2026 FKLMSTSS)

# Overview

## Trying out the Prototype

The website can be view by [download it](https://github.com/HugoXOX3/StudentUnionNexusWebsite/archive/refs/heads/main.zip) and host it locally with ```python -m http.server 80``` when python(3) environment is installed.

An alternative online prototype that is currently using can be view at [here](https://lmstnexus2526.pages.dev)

## Basic Information

There is a bilingual button for people to switch their prefer languages(English or Chinese) for a better understanding to both Chinese-speaking and English-speaking students.

Moreover,there are a map which list the restrauant near our school so that students can check it out easily by simple click and check.It is way more convinient than the traditional way of checking on instagram or leaflet and using a long period of time.

Apart from that,it can also list out the upcoming event that we are going to host.

Much more,there are more function to be updated.
